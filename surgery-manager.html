初回セットアップ
iPhoneのホーム画面から使うための設定です
1
GitHubアカウントを準備
まだアカウントがない場合は github.com/signup から無料で作成してください。
2
リポジトリを作成してファイルをアップロード
① github.com/new でリポジトリを作成
　 名前例：surgery-manager
　 Public を選択（Pages公開に必要）
② このHTMLファイルをリポジトリにアップロード
　（「Add file」→「Upload files」）
3
GitHub Pagesを有効化
リポジトリの Settings → Pages
Source: Deploy from a branch
Branch: main / root を選択して Save
数分後に https://ユーザー名.github.io/リポジトリ名/ で公開されます
4
SarafariでURLを開いてホーム画面に追加
iPhoneのSafariで公開URLを開く
下のメニュー（共有ボタン）→ 「ホーム画面に追加」
これでPWAとして動作し、通知も受け取れます 🎉
5
アプリ内から自動更新できるように設定
入力しておくと⚙️設定内「🚀 GitHubへプッシュ」でアプリを自動更新できます。
ここからToken発行（repo権限にチェック）
⚙️ 設定
🔔 アプリ内通知設定
アプリを開いている間だけ有効です
術前日（20:00）
手術前日の夜にリマインド
手術当日朝（7:00）
当日朝にチェック状況を通知
未完了タスクあり（8:00）
毎朝8時に未完了件数を通知
🧪 テスト送信
通知が届くか確認できます
🐙 GitHub自動バックアップ
データをGitHubリポジトリに自動保存します。
Personal Access Token（repo権限）を発行して入力してください。
